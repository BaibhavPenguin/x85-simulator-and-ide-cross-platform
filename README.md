# **WHAT IS x85 Simulator & IDE?**

x85 Simulator & IDE is an all in one Intel 8085A Microprocessor Simulator and Development Environment made for students. The application is targeted towards students, educators , and hobbyists who may want to explore 8085A Assembly Programming but do not have access to an 8085 Trainer Kit.

The x85 Simulator aims to bypass the cumbersome process of programming on an older platform like 8085A and make it intuitive for the user to program and monitor a simple 8085 Based Computer System

# **CORE FUNCTIONALITY** 

## USER FRIENDLY IDE

1. Features a modern and user friendly IDE for programming in 8085 Assembly Language.  
2. Instructions, Registers , Labels and Hexadecimal Data are highlighted using a consistent color code. 

## COMPREHENSIVE 8085 REGISTER AND MEMORY MONITOR

1. Features real time monitoring functionality for All the GPRs (General Purpose Registers) , The Flag Register and the individual flag bits  
2. Features a tabular monitoring interface for 64 KB of Addressable Memory with support for direct hexadecimal user input.

## PORT BASED MODEL FOR I/O

1. Upto 255 I/O ports are supported.  
2. Hexadecimal Input/Output using ports is supported.

# **8085 Programming Model**

* **Support for All 246 Unique Operations.**

* **Custom .map based executable format supported**

* **Integrated Assembler** 

* **Separate Program Listing View**

* **Support for 8085 Assembly Language with convenient assembler directives** 

# **OTHER FEATURES**

* Supports 16-Bit Hexadecimal Output Display  
* Supports 16-Bit Binary LED Array  
* v0.1 Alpha will not include support for 8085A Timing Simulation.
